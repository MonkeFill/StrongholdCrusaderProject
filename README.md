Stronghold Crusader Project is my Computer Science NEA project, it is a game based off Stronghold Crusader (2012) and created in C# using the dotnet framework and monogame, a lot of my units, UI and the way everything functions will be heavily inspired by Stronghold Crusader, lots of the assets will be taken directly off Stronghold Crusader like units, animals and tiles. All code is still made by me and assets taken from Stronghold may have slight adjusments made by myself.
